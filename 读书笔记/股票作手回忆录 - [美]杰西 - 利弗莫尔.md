# 股票作手回忆录 - [美]杰西 - 利弗莫尔

**“要不是我偶尔会赚点钱，我可能会更快就学到市场中的智慧”**



【解读】：交易这个行当最难的其实是，当你的理念有根本性错误的时候，也会时不时地赚到钱，市场中遍布各处的形形色色理念，均带有极强的迷惑性，这些都是成功路上最大的障碍，未遭到市场的沉重打击，就很难自我推翻，越晚吃亏，失败的代价只会越大，越过这些关卡是需要一定天赋的。



**“绝大多数投机者都很少见到钱。对他们来说，这些钱从来不是真实的，不是看得见摸得着的。**当一个投机者有足够好的运道将原本的资本金翻一番后，他应该立即把利润的一半提出来，放在一旁作为储备。这项策略在很多场合对我都大有裨益。”



【解读】：只有从账户里提出来的钱才是真正属于你的，用这些钱的一部分投入生活保障系统中，让他自己远离窘迫的境况。或是将提出的钱放一部分在面前，用它买下自己一直以来很想拥有的东西，它能激发你的斗志，这是一个好的心理策略，



“投资是一种艺术，股票投机尽管遵循一些主要的法则，但不仅仅只依赖数学或定理，甚至在我研读行情的时候，我所做的也不仅仅是计算，我更关心股票的行为举止，换句话说，我关注的是那些能使你判断市场是否与先例表现一致的证据，如果股票表现不妙，就不要碰它。如果找不出股票表现不妙的原因，当然你无法预知市场的方向。**无法诊断，何以预测；不能预测，自然赚不到钱。”**



【解读】：这段话已经道出了一个专业炒手正确的努力方向，或者说是在市场中学习的最佳路径：学会观察市场，仔细揣摩观察市场走势是否正常的技术，并以此为基础，掌握专业型的预测方式及方法。



**“大利润不是来自报价带，而是来自于系统全面的预判大势。我最大的缺点在于没有考虑市场的趋势，而是醉心于细小的波动。**要知道没有人能把握住所有的细小波动。”



“一个人从自己的错误中吸取教益需要很长时间。人们说事物都有两个方面。**但对股市而言，只有一方面，既非牛市的一面也非熊市的一面，而是正确的方面。”**



“记住，如果一个投资者没有自律，没有一种明确的策略，没有一个简单易行的计划，他就会陷入情绪的陷阱。”



【解读】：在投资的汪洋里，唯有让所有工具尽量简化再简化，才能稳定有效地对抗情绪的迷失。



“对我而言，我必须用赚的钱来证明自己的观点。”



【解读】：当你找到一个方法，扭转了资金曲线徘徊不前的局面，并且每一战未打开时心里已经有数~心智已趋于稳定成熟，这种感觉自己是能清楚感觉到的，只有走到这一步，才足以说明你已经开始稳步走向成功。当然，要做到这些，你必须付出最大的热情去不断思考。



“我没有多少你所想像的有趣经历。我是说，学习如何投机的过程似乎并不富于戏剧性。我失败过好几次，这当然令人不快，但我输的方式和那些华尔街上的人是一样的。**投机是一门艰苦的冒险行当，投资者必须始终关注自己的工作，否则他很快便会失业。”**



【解读】：投机这一行是艰苦的，不断地失败是你无法避开的环节，至少在你真正掌握窍门之前是这样的。好听的话是会害死人的，你必须愿意倾听深刻的 ，一针见血的不那么好听的话，有益无害。



“如果说我了解这一切太慢，这是因为我从失败中学会的。从失败到认识再到失败和认识需要时间，认识失败和找出失败的原因更需要时间。”



【解读】：学习就是不断从失败中深入思考，并逐步提升完善的过程。学会长期结合大环境跟踪分析，逐渐反推出交易的突破口。



“按自己的想法办事，我找到了自信，终于摆脱了的交易旧方法，萨拉托加事件是我最后一次靠冒险，凭运气的经历，从那以后，我开始通盘考虑基本条件而不是只抓个股，我的努力使自己在艰难的投机学习中终于升级了，这实在是漫长而艰辛的一部。”



【解读】：股市是个有机关联在一起的一个整体，只有通盘考虑整个市场的变化因素，才可能真正把握好股票价格演变的节奏，随机中蕴含着永恒的规律，不能精确描述，但可以从哲学角度深刻领会它。务必记住的是，有的时候，当你把一个问题从每个角度都研究一番，你会觉得思维更混乱了，但是别放弃，遇到这种情况，不妨将这个问题暂且一放，不再想它。过一段时间再来考虑，你会发现思路变清晰了，问题也就迎刃而解了。这就是你的“精神盲点”在发挥作用，它为你做了你没有意识到的工作！这些天才的灵敏火花并不是从你头脑中来的。通过集中自己的注意力，你就能够在自己的头脑中建立起一套关于“宇宙智慧”的完整体系，而你的灵感，正是来源于此。人们所有的天赋，所有的进步，都来自着同一个本源。



“对我而言，像在对赌行那样做交易是再自然不过了，也就是看准股价的波动趋势，抓住微小但肯定会有差价的套利（其实就是当今计算机量化自动交易的模型）。初到纽约，我仍然以此行事。没有人告诉我和过去的区别。要是有人说我的办法行不通，那我实际操作一番以检验之。因为只有一件事能说明我错了，那就是“赔钱”。投机本来就如此。”



【解读】：“看准股价的波动趋势，抓住微小但肯定会有的差价来赢利”。这就是一个世纪以前的投机天才，凭着直觉从市场里感悟出的交易真谛。



“世上乜有什么比失去你拥有的一切更能教会你什么事不该做的了，如果学会了不做某些事避免亏钱，那么你就已学会怎样赚钱，明白了吗？你开始学了！”



【解读】：人的潜能只有在面对巨大压力甚至濒临绝望的时候，才会被挖掘到极致，所有的积累会在某一刻全部汇聚发酵成洞察复杂庞大系统本质的灵感！社会的转折时期，往往是英雄辈出的时期，时代在变，这一点却从未变过。





＂健全的炒作必须以健全的交易原则为基础。大家很强调洗盘之类的旧式作法。但是我可以跟你保证，纯粹的技巧无足轻重。股票炒作跟在柜台卖股票和债券的差别，在于顾客的性质不同，而不在于诉求的性质不同。摩根公司出售债券给大众，是卖给投资人，作手散出大笔股票给大众，是卖给投机客。投资人追求安稳，为投资的资本寻求持续的投资报酬率。投机客寻求的是快速获利。作手必须在投机客当中，寻找主要的市场，投机客只要有合理的机会，能够为他的资本得到大笔的报酬，就愿意冒比正常水准高的商业风险。＂

＂股价长期下跌，真正原因绝对不是空头掼压。一支股票不断下跌时，你可以肯定其中一定有问题，不是市场有问题，就是公司本身有问题。如果下跌没有道理，股价很快就会掉到真正的价值以下，就会带来买盘，阻止跌势。事实上，空头卖出股票唯一能够赚大钱的时候，是股价太高的时候。＂

【解读】：上述两段话非常直白，对于观察判断走势是否正常有很深刻的指导意义。

＂股票炒作的基本原则是要拉抬股票以便卖出。但是你在上涨时，不能大量卖出。你办不到。大量卖出是要从头部一路下跌时卖出。……炒作的一条规定你最好记牢，就是股票要尽量炒到最高价，然后一路压低，散给大众。……就某方面来说，因为大众总是“流连”在市场中，可以断定大众经常都在亏损。投机客致命的敌人有无知、贪婪、恐惧和希望。世界上所有的法令规章、以及地球上所有证券交易所的规则，都不能把这些敌人从人的兽性中消除。＂

【解读】：在价格的惯性起落中，人性的贪婪暴露无遗，这里面有着永远都不会改变的规律，价格总是在伴随着一定的粘滞度中前行。对于作手而言，只要营造出丰富的前戏，那么随后在暴跌中搏反弹的贪婪，总能为自己出脱大量的股票提供足够的买盘。

＂我犯了个错误，但错在哪儿呢？在熊市遵循熊市的作法，这是英明之举。我做了空头，这也是正确的。我抛得太早，代价太大，我的头寸是正确的，但我的时机错了。……行情记录还是要研究的，这样才能判断出行动的适当时机。如果开始时你做对了，那么你所处的有利情况就不会受到威胁，然后你会发现抱牢手上的头寸，也就不再困难。……我看对了，但是，我破产了！＂

【解读】：看对和做对～严谨把握好进场时机，都是至关重要的，缺一不可。毕竟利弗莫尔是人而不是神，他的经验很有价值是面镜子，功绩在于他坦诚自己的所有失败细节，让后人少走弯路，这才是根本。个人认为，就这个典型案例来说，问题在于，求胜太心切，第一是过早进场，这个可以用有计划地分批进场来解决，第二是犯了重仓大忌，违反了概率法则，我个人认为，投资的概率法则是只要不是100%的胜算(个人投资者基本都是如此)，就绝不可孤注一掷，否则就极有可能应了墨菲定律，最不可能出现的最糟糕情况往往就在此刻出现了，这也是科学的资金管理规则！在人工交易中个人的信念到底是什么，是想战胜市场，还是要稳健地赚钱，不同的人的区别就在这个瞬间体现了。

＂经纪商只关心赚取今天的手续费，因此一般证券商出版的杂志有着无法避免的错误推断。证券商靠大众的手续费赚钱，可是他们会设法利用他们的杂志，引诱大众，购买他们从内线人士或作手收到卖单的同一只股票。＂

【解读】：在市场之外，还有大量的陷阱等待着普罗大众，警惕一切＂送钱＂的消息和建议，天下没有白吃的午餐，券商和内线人士有着纠缠不清的利益关系。

"内线人士可能告诉大赌客买进的时机，但一定说不出什么时候是卖出时机！......有些内线人士在这种游戏的任何阶段，你都不能信任。"

【解读】：出货是需要在价格的各阶段性高点人为地创造出大量的需求，太多的谎言都是为这个核心目的服务，这些消息信号弹的发射是投机世界里永恒的＂黑色艺术＂，因为人性不变，计策万变也不离其宗。

＂我既没有一种僵硬的系统，也不遵照这种系统操作。我根据状况，随时修改自己的条件。＂

【解读】：金融市场只存在交易逻辑的圣杯，而绝不存在交易技术或交易系统的圣杯！交易逻辑是拥有一定柔性的科学的合理的市场思维体系：

\* 柔性的——交易系统是交易逻辑与特定时期、特定市场大环境相结合的产物，必须是动态的。

\* 科学的——任何有价值的投资理念必须具备数学上的可推演的逻辑基础。

\* 合理的——遵循市场的基本供需原理。

------



书里的思想可以不断穿透新的时代，的确，从本质上说，华尔街无新鲜事，投资如山岳般古老，在信息技术爆炸的现代社会仍然如此，术千变万化，道不变。

对于阅读经典，要注意的是，投资的世界里，学习他人不可一切照搬，盲目克隆他人的术，那是不可能成功的，甚至会导致惨败。针对不同的市场环境，道相同，术可以完全不同，在那个没有计算机的年代，为了精细了解市场，利弗莫尔会动用人力进行大量人工计算统计供自己操盘决策，现在这个繁杂工作可以交给计算机完成，计算机功能强大，但是它没有思想，观察市场的思维和灵感需要靠自己的聪明才智，这也是我们需要反复对先辈炒家们进行深入学习思考的原因。

在做事的专注度上，我无比尊敬利弗莫尔，纵使他是交易天才，在交易上一直保持孩童般的探索精神，非常认真严肃自己的每一个失败细节，将自己的整个生命融入了交易，真正当做事业在做。时代在变，游戏的规则在变，用当今的市场环境去批判他的操作细节，这种钻牛角尖的做法是很可笑的，另外没有人有能力达到他当年的高度，纵然拼搏一生也难望其项背。从他的成功和失败特别是失败中去深入思考学习，才是可取的做法。作为现代新巿场环境下的交易者，具体交易方法和系统与他完全不同是很正常的，但是市场的根本原理及规律永恒不变，对市场的解读和投资的核心思维是相通的，利弗莫尔的理念在当今仍然是极有价值的。

对此，利弗莫尔说过：＂我对自己事业的每个阶段都深感兴趣。我当然从别人和自己的经验中学习。但是今天很难从下午收盘后，流传在号子里的故事中，学习如何炒作股票。当年大部分的手段、绝招和妙招都已经过时、没有用了，或者是非法、不能再用了。证券交易法令和情势已经变了，丹尼尔·朱鲁或小雅各或古德在50到70年前能做的事情，即使是精确详细的事迹，也没什么听的价值了。今天的作手不必考虑这些前辈所做的事情，或是考虑他们怎么这样做，就好比西点军校的学生不必学习古人的箭术，才能增加实用的弹道学知识。另一方面，研究人性因素总会有好处，例如：为什么人这么轻易相信自己希望相信的事情？为什么他们让自己——其实是鼓励自己——受贪心的影响，或是受一般人粗心大意、斤斤计较的影响。恐惧和希望总是相同的，因此，研究投机客的心理，其价值始终如一。武器会改变，但是战略还是战略，无论在纽约证券交易所，还是在战场上，都是这样。我认为对于整个情形最能执简御繁的话，是汤玛斯·伍罗克所说的，他说：'股票投机成功的基础，是假设大家未来会继续犯以前所犯的错误。'＂。



------



千人有千悟，不必太追究书里面的具体方法和规则，应该关注和反复思考的是利弗莫尔观察市场的思维方式的＂道＂，那正是投资成功之道。

投资的路注定是艰辛和寂寞的，成功来之不易。艰难的时候你会看尽世态炎凉，人性的丑陋会赤裸裸地展现在你面前，而这些东西带来的挫败感会撕毁心中的诸多美好，那种孤独寂寞的感觉，会逐渐让人的精神世界变成一片荒漠，寸草不生。这是一个人的天荒地老，反复独自徘徊地狱和天堂之间，在赢得金钱的同时注定会失去太多，我想这应该就是利弗莫尔最终饮弹自尽的真正原因吧。

从利弗莫尔的一生，我们可以得到一个启示，人的一生要讲究一种平衡，追逐梦想释放自己和充实生活的一种平衡，过于疯狂地追逐梦想追逐事业，注定燃尽自己，过后会发现，自己最终其实什么都没真正战胜，然而快乐早己离自己远去。人这一生的快乐是有定数的，过度透支和挥霍，前方不远处，无尽的抑郁正等着你。

欧奈尔曾告诫说：“在生活中做一个‘无聊’的人，在市场中做一个敏锐者，你就能避免利弗莫尔的精神痛苦”。认识你自己，淡定过好这一生，其实并不难，只要你愿意让自己接受占生活大部分时光的平淡。

事业和生活，讲究适可而止，懂得到了一定程度及时收手，回归平静的生活，才是真正智慧的人生。

“持而盈之，不如其已；揣而锐之，不可长保。金玉满堂，莫之能守；富贵而骄，自遗其咎。功成身退，天之道也。” ---- 老子
