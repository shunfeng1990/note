# 多进程-多线程-协程

> 进程是资源单位，线程是执行单位

#### 多线程实现

```python
"""
 多线程实现 唱歌跳舞同时执行
"""
from threading import Thread  # 导入线程类


def sing():
    """
    唱歌函数
    """
    for i in range(5):
        print('唱歌中')


def dance():
    """
    跳舞函数
    """
    for i in range(5):
        print('跳舞中')


def main():
    t1 = Thread(target=sing)
    t2 = Thread(target=dance)
    t1.start()
    t2.start()


if __name__ == "__main__":
    main()
```

#### 多线程另外一种写法

```python
from threading import Thread  # 导入线程类


class Mythread(Thread):
    def run(self):  # 固定的 线程被执行的时候 执行的就是run()
        for i in range(5):
            print('子线程')


def main():
    t = Mythread()
    t.start()


if __name__ == "__main__":
    main()
    for i in range(5):
        print('主线程')
```



#### 多进程实现方法（跟多线程基本一样）

```python
#! /usr/bin/python3
from multiprocessing import Process  # 导入线程类


def test1():
    while True:
        print('进程1')


def test2():
    while True:
        print('进程2')


if __name__ == "__main__":
    p1 = Process(target=test1)
    p2 = Process(target=test2)
    p1.start()
    p2.start()
```



#### 线程池 和 进程池

> 一次性开辟一些线程，把任务交给线程池去做，至于怎么调度那是线程池的事了。
>
> 进程池也是一个样。

```python
from concurrent.futures import ThreadPoolExecutor, ProcessPoolExecutor


def test1():
    for i in range(10):
        print('test1')

def test2():
    for i in range(10):
        print('test2')


if __name__ == "__main__":
    # 创建线程池
    with ThreadPoolExecutor(5) as t:  # 开辟5个线程
        t.submit(test1)  # 把任务扔给线程池
        t.submit(test2)
    print('线程池执行完毕在执行这里')
```
